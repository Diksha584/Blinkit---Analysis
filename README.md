# 🛒 Blinkit Sales Analysis

## 📌 Project Overview
This project analyzes Blinkit grocery sales data using **Python, SQL, and Power BI**. The objective is to clean the data, perform exploratory data analysis (EDA), and build an interactive dashboard to identify sales trends and business insights.

---

- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL Server
- Power BI

## 💡 Key Insights
- Tier 3 outlets generated the highest sales.
- Regular fat products contributed more sales than low-fat products.
- Fruits & Vegetables and Snack Foods were the top-selling categories.
- Supermarket Type 1 recorded the highest overall sales.
